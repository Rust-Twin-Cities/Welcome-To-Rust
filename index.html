Welcome to Rust

A whirlwind introduction

What is Rust?

A systems programming language pursuing the trifecta

  • Safe
  • Concurrent
  • Fast
It's a functional programming language (impure) that will feel familiar to C/C++ developers. It's like if C++ and Haskell had a baby that grew up to date Ruby and Python It's called Rust, by the way, because none of the ideas in it are new.

What is Rust for?

Large scale, low level systems: browsers, servers, operating system components, databases, games, basically anything you might write in c++. note that is is what rust was built _for_ not what it _can do_ it's turing complete so... do what ever you want. But that doesn't necessarily mean it's appropriate for everything.

Installation

Initially I was going to make this a Vagrant based install, but I watched this keynote where they talked about windows being second-class citizens in python and how this was detrimental to the ecosystem. So let's try to make Windows first class citizens in Rust.

Unix-y Systems

Run this in a terminal

curl -s http://www.rust-lang.org/rustup.sh | sudo sh
You may need to substite in zsh or whatever shell you are using.

Windows

Windows is currently a second class citizen!

  • Download Virtualbox
  • Download Vagrant
  • Run the installers with default options
  • Wait a year and half
  • Restart
  • Start the Command Prompt (search for cmd.exe)
  • Type "vagrant init"
  • then "vagrant up"
  • Finally type:
    • curl -s http://www.rust-lang.org/rustup.sh | sudo sh

This should get you:

  • rustc - the Rust compiler
  • cargo - the Rust package manager

Let's test it out

Open up your favorite text editor

Type out:


fn main() {
    println!("Hello World");
}

Save it as "hello_world.rs" somewhere memorable

Compile it

Open up a terminal and run:

rustc "path/to/hello_world.rs"

Who got errors?

Errors I've encountered

  • could not load shared library...
  • command rustc not found

Make it a package

  • move hello_world.rs to src/hello_world.rs
  • Create a file Cargo.toml in the same directory as hello_world.rs
  • Add the configuration:

                        [package]

                        name = "hello_world"
                        version = "0.1.0"
                        authors = [ "someone@example.com" ]

                        [[bin]]

                        name = "hello_world"

Use cargo

Run this in the current directory:


                        cargo build

Explicitly Returning


fn factorial(i: int) -> int { 
    let mut counter = i;
    let mut sum = 1;
    loop {
        if counter <= 1 {
            return sum;
        }
        
        sum *= counter;
        counter -= 1;
    }
}
